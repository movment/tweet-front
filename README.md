# tweet

---

tweet은 트위터를 클론하여 개발중인 프로젝트입니다.

## Quickstart

---

깃을 클론한 뒤 해당 프로젝트 내부에 .env.local 파일을 만듭니다.

그리고 .env.local에 JWT_KEY=your_key를 작성합니다.

#### Install

```
$ npm install
```

#### build

```
$ npm run build
```

#### start

```
$ npm run start
```
